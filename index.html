Git

  • The repository is where all of the code for the web app is located on GitHub
  • Never work off of the main branch of the repository
  • Always make a new branch for all of the changes you are going to make
  • git checkout -b branch-name: makes a new branch and switches to it
  • git status: checks what branch we are currently on
  • Once you have made the changes you want in your branch, you can submit a pull request to request the changes in the branch you made are pulled into the main branch
  • Once your pull request has been reviewed and accepted, it will be merged into the main branch
